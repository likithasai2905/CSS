CSS - Cascading Style Sheet:-

CSS is the language used to describe the presentation of HTML and to style the web pages [like Colors, Sizes, Spacing, Layout, Animations, accessibility hints , etc].

RWD full form is Responsive Web Design:It is the approach of resolutions,and device capabilities.

3-ways of styles to insert to html:

1.Inline Styling

2.Internal Styling

3.External Styling

1.Inline Styling:-In css we written directly in the HTML element using the style attribute.

2.Internal Styling:-It is used to write the code inside the style tag in the Html files head section.

3.External Styling:-css is kept in seperate .css file and it is linked to the html using link tag inside the head tag.
